## Overview 🌟
Welcome to my Python learning repository from **Coding Factory 9**. This is a hands-on collection of scripts and examples I'm building as I learn Python, focused on practical, real-world snippets and exercises that reinforce each concept.

## About the Repository 📖
This repo takes a practical approach to learning Python, working through core language features one small, focused example at a time. Each lesson lives in its own folder with the source code and a detailed README explaining the concept, the code, and a few exercises.

## Repository Contents 📂
### Practical Applications 🛠️
- Data Manipulation 🔢
- User Interaction ⌨️
- Basic Algorithms 🔄
- Mathematical Demonstrations 📏

## Getting Started 🚀
No installation is required beyond Python itself:
1. Ensure Python 3.x is installed on your machine.
2. Clone the repository: `git clone https://github.com/jroumpekas/cf9-python.git`
3. Navigate to the lesson folder of interest inside `chapter01`.
4. Open a terminal or command prompt.
5. Run a script with `python <filename>.py`, and read its README for an explanation.
